# Yaseen Dada Shaik
### Machine Learning Engineer · Research-Grounded · Production-Ready

I work across the full depth of ML — from mathematical foundations and research paper
reproductions to custom architecture experiments and production systems serving real users.

My approach is first-principles: I build models from scratch before using libraries,
work through the mathematics by hand before implementing, and read papers before
using techniques. Then I take that understanding into production.

---

## What I Work On

**Research & From-Scratch Implementations**
- Language model architectures built from scratch in PyTorch (MLP, n-gram, character-level)
- Backpropagation and optimization implemented without autograd
- Foundational ML paper reproductions — understanding internals, not just running code

**Custom Architecture Experiments & ML Systems**
- Building and experimenting with custom model architectures from scratch
- Designing novel training pipelines and optimization strategies
- Dockerized model serving and deployment pipelines
- Benchmark evaluations using AirLLM and related frameworks
- Scaling experiments: memory efficiency, throughput, latency tradeoffs

**Production ML Systems**
- End-to-end transformer fine-tuning pipelines (DeBERTa, HuggingFace)
- Inference optimization: ONNX export, INT8 quantization, CUDA training
- ML serving via FastAPI microservices with model versioning
- Continual learning with replay-based training

**Agentic & LLM Systems**
- Multi-agent orchestration runtime design
- RAG pipelines, LLM orchestration, MCP integration
